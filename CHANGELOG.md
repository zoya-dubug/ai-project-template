# Changelog

All notable changes to this project template will be documented in this file.

This project follows a version-based development approach. New features and structural changes are introduced only after they have proven useful across multiple real-world projects.

---

# [v1.0] - July 2026

## 🎉 Initial Release

### Added

- Minimal AI project directory structure
- Professional `README.md`
- `NOTES.md` template for project planning and experiment tracking
- Generic `requirements.txt`
- Standard `.gitignore`
- MIT License

### Design Philosophy

- Start simple.
- Expand only when the project requires it.
- Avoid unnecessary folders and boilerplate.
- Keep the template lightweight and maintainable.

### Frozen Structure

```text
AI-Project-Template/
│
├── .gitignore
├── LICENSE
├── NOTES.md
├── README.md
├── requirements.txt
│
├── data/
├── models/
├── notebooks/
├── results/
└── src/
```
